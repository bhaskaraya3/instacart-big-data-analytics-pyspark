# INSTACART BIG DATA ANALYSIS

---

A Big Data Analysis using pyspark over **37M+** rows.

---

## Business Questions :-

1. Department with the max item sold
2. Aisle with the max item sold
3. Product sold the most
4. Department wise most product sold --> further product name
5. Which day of week has max order and min order too
6. Which hour of the day has max order and min order too
7. Which product added at cart first --> department --> aisle --> product name
8. Which product added at cart last --> department --> aisle --> product name
9. Those product which are added to cart first, are those ones reordered the next time by the customers?
10. Which customers has the minimum days since prior order --> get to know our loyal and repeating customers
