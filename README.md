# 🛒 Instacart Customer & Product Analytics (PySpark)

## Project Overview

This project analyzes over **37M+ Instacart orders** using **PySpark** to uncover customer behavior, product demand patterns, and actionable business insights.

The goal is to simulate **real-world retail decision-making** using data.

---

## Tech Stack

* PySpark (Big Data Processing)
* Python
* Jupyter Notebook
* Data Engineering + Analytics

---

## Project Highlights

* Processed and analyzed 37M+ rows using PySpark
* Built feature-engineered dataset for behavioral analysis
* Identified customer segments and demand patterns
* Derived actionable business insights for retail optimization
